02 — Security & data protection

Bank-grade by design, not by checkbox

Tenant isolation, signed credentials, and a complete audit trail — built specifically for the accountability that recovery work demands.

Per-agency data isolation

A genuinely separate database per agency (crmr_<agency>) — one agency can never see or touch another's records.

Token-based auth, everywhere

HMAC-signed access tokens route apps only to their own database. No request reaches data without a valid, signed credential.

Encrypted transport (TLS 1.3)

All traffic runs over HTTPS / TLS 1.3 with auto-renewing certificates. Nothing travels in plain text over the network.

Device-locked agent logins

Each login binds to a single device. A device change requires an OTP plus admin approval, so a lost phone can't sign in elsewhere.

Email OTP verification

Agency registration and admin sign-in are protected by one-time codes — no password-only access to sensitive areas.

Anti-spoofing email

Outgoing mail is signed and verified with SPF + DKIM + DMARC, so notifications can't be spoofed and reliably reach the inbox.

Role & finance-level access

Admins and agents hold different rights, and agents can be restricted from specific financers — they only see what they're authorized to.

Server-enforced controls

Accounts can be activated, stopped, or blacklisted instantly, and subscription expiry is enforced server-side — checked continuously, not just at login.

Full audit trail

Who searched, which vehicle, where, and when — captured with both device and server time, and encrypted mailbox storage at rest.

Database

Access

Speed

Two purpose-built apps, one source of truth

Head office commands from the Windows console; agents work the field from a white-labeled Android app — kept in perfect sync.

Head-office admin

Windows Desktop Console

Run the whole operation from one screen — search, finances, confirmations, live map, and reports.

  • Instant search by RC or chassis, plus "Found in N finances" across every financer & branch.
  • Finance & branch management — head offices, branches, contacts, addresses.
  • Bulk Excel upload with column-mapping, validation, duplicate handling & replace.
  • Confirmations workflow — Confirm / OK-for-Repo / Cancel with WhatsApp & SMS.
  • Live users map — real-time GPS location of every field agent.
  • Search & audit logs, exportable to Excel/PDF and filterable by date or agent.
  • User management — agents, admins, device-change approvals, blacklist.
  • Subscriptions, support tickets, dashboards & reports — in a per-agency branded installer.
Field agent

Android Agent App

Offline-first and white-labeled per agency — instant search even on the weakest network.

  • Instant search by RC/chassis — works online and fully offline.
